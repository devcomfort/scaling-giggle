Online LMS Downloader

2021년 2학기 들어 LearningX의 영상 다운로드가 막힘에 따라, 온라인에서 영상을 다운 받을 수 있도록 페이지를 만들었습니다.

이 페이지를 사용해서 생기는 피해에 대한 책임은 모두 사용자에게 있습니다. ➡️ 대충 유포하지 말고, 저작권 지키라는 소립니다.

 

사용법

  1. 러닝엑스에서 영상 주소를 가져옵니다. 영상 주소를 찾는 방법은 다음과 같습니다.

    1. 다운받고자 하는 영상의 페이지에 들어가서 Ctrl + J 혹은 F12 키를 눌러 개발자 모드를 활성화합니다.

    2. Ctrl + Shift + C를 누르거나, 좌측 상단의 아이콘을 눌러 inspector 기능을 활성화 합니다.

    3. 동영상 화면을 클릭합니다. 2. 과정을 했다면, 아래 마우스가 영상 화면에 올라가는 순간 아래 화면과 같은 모습이 됩니다.

    4. video 태그의 링크 부분을 복사합니다.

    5. 받아온 영상 주소를 다음과 같이 서버에 입력한 후 기다립니다. 예시:

      링크 입력 시 규칙/세부사항

      • ;을 통해 파일 이름을 별도로 지정할 수 있습니다. 예시: https://abc.com/video/1/2/3/video.mp4;간호용어1.mp4 이 경우 video.mp4라는 이름을 가졌던 파일이 간호용어1.mp4라는 이름으로 다운로드 됩니다.

        별도의 설정이 없는 경우 기본값인 video.mp4로 다운로드 됩니다.

      • 유효하지 않은 URL은 유효한 URL이 모두 처리된 후, 입력 공간에 남습니다. 경고창에 유효하지 않은 URL들과 함께 사유 또한 같이 표기되니 참고해서 사용하시기 바랍니다.
      • 다운로드 위치는 변경할 수 없습니다. JavaScript를 짜는 과정에서 여러번 확인했지만, 현재 과정에서는 처리할 수 없었습니다.😢

    6. 이제 다운로드 중... 문구가 사라질 때까지 기다리시면 됩니다.

 

또 다른 개발자에게

웹사이트 특성상 Ctrl + U를 통해서 사이트 코드를 볼 수 있습니다. 그것을 통해서 업데이트/수정을 하려는 시도가 있을 것 같은데, 그냥 아래 주소로 들어가면 Github Repo 주소를 볼 수 있습니다. https://github.com/devcomfort/LMS-Downloader-Online

본 서버는 Heroku를 통해 서비스되고 있으며, Go언어의 Gin 모듈을 사용해 정적 파일을 서비스 합니다. 하지만 위의 Github Repo에는 Go언어는 볼 수 없을겁니다. 이 프로그램의 핵심인 정적 파일만 넣어뒀으니 참조바랍니다.
